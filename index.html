Stone Block 2 Guia

Guia em Português de objetivos do modpack StoneBlock 2.

Objetivo 1:

Crafite um crook para obter folhas de uma muda

Objetivo 2:

Colete terra de 9 folhas (ou peneirando cascalho)

Objetivo 3:

Faça um martelo e use-o para quebrar pedra e coletar cascalho

Objetivo 4:

Peneire o cascalho para conseguir recursos

Objetivo 5:

Faça uma pequena mob farm para começar a coletar loots de mobs

Objetivo 6:

Faça um power core, ele irá armazenar energia que podera ser usada até 30 blocos de onde o power core estiver

Objetivo 7:

Faça um combustion generator ou um lava generator para conseguir gerar energia

Objetivo 8:

Comece a plantar sementes para conseguir alimentos

Objetivo 9:

Faça um wooden barrel e coloque folhas nele para gerar agua

Objetivo 10:

Coloque pedra em um stone barrel para gerar lava

Objetivo 11:

Faça um cobblestone generator e coloque-o dentre 30 blocos de distância do power core para fazer pedra

Objetivo 12:

Coloque um fúnil em cima do cobblestone generator para coletar cobblestone automaticamente

Objetivo 13:

Faça um solar panels ou um water wheel para começar a gerar energia passivamente

Objetivo 14:

Faça spikes e coloque-os na sua farm mob, use funis ou carrinhos com funis para coletar os drops dos mobs

Objetivo 15:

Faça uma região com terra para animais

Objetivo 16:

Faça um crusher para transformar cobblestone em cascalho automaticamente, use conveyors para mover a pedra para o crusher

Objetivo 17:

Faça uma auto sieve para conseguir peneirar mais facilmente

Objetivo 18:

Atualize a sua geração de energia ate ele gerar pelo menos 200 RF / s

Objetivo 19:

Faça um void miner para coletar minerios do void

Objetivo 20:

Faça uma mechanical furnace para transformar os blocos de minerios em minérios automáticamente

Objetivo 21:

Faça um auto farmer para automatizar a sua fazenda

Objetivo 22:

Faça uma plant forge para fazer tipos variados de sementes

Objetivo 23:

Use um drying rack para conseguir couro usando carne podre

Objetivo 24:

Faça uma mesa de encantamento

Objetivo 25:

Faça uma salvage table e uma create table, esses itens irão permitir que você faça trocas por outros itens

Objetivo 26:

Consiga um full set de armadura de diamante encantado

Objetivo 27:

Use um matter converter parar converter areia em areia da alma

Objetivo 28:

Peneire areias da alma para conseguir ender pearls e varas de blaze

Objetivo 29:

Crie uma empty ring e combine-a com uma poção para fazer potion ring

Objetivo 30:

Faça um end cake e derrote o ender dragon

Objetivo 31:

Pegue a cabeça e o ovo do dragão

Objetivo 32:

Faça um dragon egg generator e coloque o ovo do dragão em cima dele

Objetivo 33:

Atualize a sua geração de energia ate gerar no mínimo 2000 RF / s

Objetivo 34:

Vá para o end e colete draconium ingots de end ores

Objetivo 35:

Faça um draconic ifuser e drope um draconium ingots nele isso quando você tiver armazenado 100,000 RF

Objetivo 36:

Faça um exoskeleton usando draconium ingots

Objetivo 37:

Coloque seus exoskeleton em e fique em cima do draconic infuser parar cobrar isso, faça um draconium battery para checar seu exoskeleton power

Objetivo 38:

Mate alguns robos no end e use as scrap parts para fazer um novo torso, arm, e leg(peitoral, braço e perna)

Objetivo 39:

Faça um power charm, isso dará a você um monte de efeitos isso junta com outras partes de android, exoskeletons, e baubles

Objetivo 40:

Faça um overenchantment table e comece a fazer alguns encantamentos customizados

Objetivo 41:

Use cristais do fim para reviver o ender dragon e coletar outra cabeca de dragão

Objetivo 42:

Faça um draconiumcannon ou um awakened draconium cannon, isso irá usar o poder dos seus exoskeleton para atirar explosive projectiles

Objetivo 43:

Faça um set de armadura full encantado de netherite

Objetivo 44:

Invoque e mate o wither

Objetivo 45:

Faça um lucky block usando 8 ouros e abra-o (o lucky block não aparece na craft in table)

Objetivo 46:

Faça um fisher e coloque-o na agua para pescar peixes automáticamente

Objetivo 47:

Faça um storage unit e use conveyors para colocar itens nele, os itens sao coletados e guardados no power core

Objetivo 48:

Faça um passive hostile, ou nether spawner e faça uma farm de mob automática

Objetivo 49:

Faça um copper e coloque-o em baixo de um bloco de terra com uma árvore em cima cara cortar a arvore automáticamente

Objetivo 50:

Explore o as ilhas do end e consiga 4 elytras para crafiar um angel charm
